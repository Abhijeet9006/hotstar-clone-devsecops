# Hotstar Clone DevSecOps CI/CD Pipeline

This project implements a secure CI/CD pipeline for a Hotstar Clone application using DevSecOps practices.

## Tech Stack

- React
- Docker
- Nginx
- Jenkins
- SonarQube
- Trivy
- Terraform
- AWS EKS
- Kubernetes
- OWASP ZAP

## Project Source Acknowledgement

The sample Hotstar Clone application is based on the public repository:

https://github.com/DevOpsInstituteMumbai-wq/Implementing-a-Secure-CI-CD-Pipeline-for-Hotstar-Clone-Using-DevSecOps-Principles

This repository extends the application with Dockerization, CI/CD automation, security scanning, infrastructure provisioning, and Kubernetes deployment.

## Completed So Far

- React app setup
- Local build verification
- Docker image creation
- SonarQube SAST scan
- Trivy container scan
- Jenkins installation and configuration
